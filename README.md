# 🚀 Placement Preparation Hub

Placement Preparation Hub is a web-based platform designed to help students prepare for campus placements in one place. The platform provides aptitude preparation, coding practice, AI-based doubt solving, online compiler support, interview preparation resources, and curated learning content.

The main goal of the project is to make placement preparation simple, organized, and accessible for every student.
